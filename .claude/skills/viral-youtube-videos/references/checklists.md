# Fast pre-publish checklists

Quick sanity checks. Any "no" is a fix-it, not a nitpick. Full reasoning lives in
the other reference files.

## Idea checklist
- [ ] Is it **broad** — appealing to *most* of the niche, not a narrow sub-topic?
- [ ] **Fresh-perspective:** still exciting a couple of days after writing it down?
- [ ] Has **no one already** made it (or can I clearly beat their title/thumbnail/twist)?
- [ ] Is it the **best** idea I have right now (aiming ~5–10× outlier, top ~10%)?
- [ ] Can I make a **title + thumbnail** that instantly connects audience motivation → this video's value? *(If no → kill the idea.)*

## Title & thumbnail checklist
- [ ] Is there a **reason to watch apparent in under 1 second**?
- [ ] Do title and thumbnail **build on each other** (not repeat the same info)?
- [ ] Is there a **spotlight element** (face / familiar object / short text) that pops on the whole page?
- [ ] Is it **1-second simple** and **high contrast** (brightness *and* color)?
- [ ] Is the thumbnail **made from scratch** (not a screenshot) and does it *represent* the idea?
- [ ] Is it **perfectly clear** (no confusion) while still creating **uncertainty/intrigue**?
- [ ] Does the title lead with the **most important words** and read as a **driving question** (or clear context)?
- [ ] How many **reasons to watch** does the pair stack? (More = more robust.)

## Hook / intro checklist
- [ ] Does it **deliver the promised content** from the title/thumbnail, **fast**?
- [ ] Does it deliver the promised **feeling** (pacing / mood / style)?
- [ ] Is a **reason to watch** established before any payoff?
- [ ] Is **every sentence** meaningful (rewrite up to 5× if not)?
- [ ] Do visuals / SFX / music **change every few seconds** (not a static talking head)?
- [ ] Does it **exceed** expectations — tease extra value / the coolest part?
- [ ] **No** asking for likes/subs/money.

## Retention / script checklist
- [ ] Is the **most extraordinary thing near the start** (proof > promise)?
- [ ] Does **every part** have a green "reason to watch" before its purple payoff?
- [ ] At each stretch, is it clear **what the viewer would miss by leaving here**?
- [ ] Is anything **confusing**? (Confusion is the #1 killer — a fresh viewer test helps.)
- [ ] Is the structure **but/therefore**, not **and then**?
- [ ] Does **every point** have a supporting **visual**, with focus guided (zoom/pan)?
- [ ] **Mood variety** present; **visual variety** not overdone (seasoning, not a dump)?
- [ ] Is it **compact but conversational** (not bloated, not over-trimmed)?
- [ ] Is the **second half as strong as the first** (binge-able, no outro)?

## Edit checklist
- [ ] Does the **editing style match the experience** the audience came for?
- [ ] Is **pacing as fast as the content allows** (benchmark ~5s/clip even when slow)?
- [ ] **Visual variety** (a-roll short/important, b-roll heavy, motion graphics for hard explains) without over-cutting into mush?
- [ ] **Visual continuity:** graphics animate/sound in; cuts keep the focus point consistent?
- [ ] **Sound design:** whooshes/highlights added; risers/hits/drones used sparingly and only when earned?
- [ ] **Music:** mood matched per segment, hits synced, pauses/fades used for emphasis?
- [ ] **Energy varies** (hyper-edited → mellow) to force re-focus?
